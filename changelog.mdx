## Latest Updates

Deepika Nallana evolves rapidly to help you manage project documentation more effectively. Review the most recent releases below, ordered newest first. Each version includes new features, improvements, and bug fixes.

## Upgrade Guide

Upgrading Deepika Nallana ensures you benefit from the latest features and security patches. Follow these steps for a smooth transition.

<Steps>
  <Step title="Backup Your Data" icon="database">

    Export your documentation space.

````bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.example.com/v1/export?spaceId=YOUR_SPACE_ID \
  -o backup.zip
````

  </Step>
  <Step title="Update Client SDK" icon="download">

    <Tabs>
      <Tab title="npm" icon="package">
````bash
npm update @deepika/sdk@latest
````
      </Tab>
      <Tab title="yarn" icon="package">
````bash
yarn upgrade @deepika/sdk@latest
````
      </Tab>
    </Tabs>

  </Step>
  <Step title="Run Migrations" icon="settings">

    Execute the migration script.

````javascript
// migration.js
const sdk = require('@deepika/sdk');
await sdk.migrate({ version: '1.2.0', spaceId: 'YOUR_SPACE_ID' });
console.log('Migration complete.');
````

  </Step>
  <Step title="Verify" icon="check-circle">

    Test key features in your dashboard.

  </Step>
</Steps>
