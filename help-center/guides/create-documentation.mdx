---

## Introduction

Start documenting your project in a few steps.

## Steps

<Steps>
  <Step title="Go to project space">
    Log in and select your project from the dashboard.
  </Step>
  <Step title="Create a new page">
    Click the "Add Page" button and choose a title.
  </Step>
  <Step title="Add content">
    Use the editor to add text, images, or code snippets.
  </Step>
  <Step title="Save">
    Click "Save" to publish your new documentation page.
  </Step>
</Steps>
